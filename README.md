# Currency Converter - Конвертер валют

**Автор:** Вакулин Александр

## 📝 Описание

Currency Converter - это графическое приложение для конвертации валют в реальном времени с использованием внешнего API. Приложение позволяет конвертировать суммы между различными валютами, просматривать актуальные курсы и сохранять историю конвертаций.

## ✨ Основные возможности

- 💱 Конвертация между 16 основными валютами
- 📊 Актуальные курсы валют через API
- 📜 История всех конвертаций с сохранением в JSON
- 🔄 Автоматическое обновление курсов
- 💾 Экспорт истории в JSON файл
- 🗑 Очистка истории
- ✅ Валидация ввода (положительное число)

## 🔑 Как получить API-ключ

Приложение использует бесплатный API [ExchangeRate-API](https://app.exchangerate-api.com/):

1. Перейдите на сайт: https://app.exchangerate-api.com/sign-up
2. Зарегистрируйтесь (можно через Google или email)
3. После регистрации вы получите бесплатный API ключ
4. Скопируйте ключ
5. В файле `main.py` замените `YOUR_API_KEY_HERE` на ваш ключ:
